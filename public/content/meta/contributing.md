# Contributing

Thank you for your interest in contributing to the Slayer Legend Wiki! This guide will help you get started.

## Getting Started

The Slayer Legend Wiki is a community-driven resource built using GitHub. All contributions are welcome, whether you're fixing typos, adding new content, or improving existing pages.

## Ways to Contribute

### 1. Edit Existing Pages

1. Navigate to any page on the wiki
2. Click the **Edit** button (requires GitHub login)
3. Make your changes in the editor
4. Add a brief summary of your changes
5. Click **Save Changes** to create a pull request

### 2. Report Issues

If you notice errors or have suggestions but don't want to edit directly:

1. Visit the [GitHub repository](https://github.com/BenDol/SlayerLegendWiki)
2. Open a new issue describing the problem or suggestion
3. Include page links and specific details

### 3. Add New Content

To add new pages:

1. Create a new markdown file in the appropriate section directory
2. Add frontmatter with title, description, and tags
3. Write your content using Markdown
4. Submit via pull request

## Content Guidelines

- **Be accurate**: Verify information before adding it
- **Be clear**: Write in simple, easy-to-understand language
- **Be respectful**: Follow our community guidelines
- **Cite sources**: Link to official sources when possible

## Markdown Tips

The wiki supports GitHub-flavored Markdown:

```markdown
# Headers with #
**Bold text**
*Italic text*
[Links](https://example.com)
![Images](/images/example.png)
```

## Getting Help

Need assistance? Check out our [Editing Guidelines](/#/meta/guidelines) or reach out through GitHub issues.

Thank you for contributing to the Slayer Legend community!
